      - >1.SaaS的本质是“续费”，估值按市销率的10倍，传统是按市盈率的20倍。2.SaaS产品最适合“橄榄型”（营收占比维度）的目标市场，市场分析（波特五力模型）3.SaaS相关的商业模式：买断，收年费，消耗，分销。4.四类营销人才：toB解决方案销售，toB快单销售，to小B门店地推，toC客户是个人。5.越是快单、短周期销售，要求的标准化程度越高；大客户销售标准化程度要求相对较低。但即便是大的销售解决方案，也需要“标准的流程”。销售全流程：蓄客，验证，邀约，首次拜访KP，再次拜访，签约回款，转介绍，转交服务。6.销售团队的搭建与组织管理。7.SaaS公司财务模型。
